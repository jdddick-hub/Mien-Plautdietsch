# Mien Plaut

Statische Website für GitHub Pages.

## Veröffentlichung

1. Auf GitHub ein neues Repository erstellen, zum Beispiel `mien-plaut`.
2. Alle Dateien aus diesem Ordner direkt in das Repository hochladen.
3. Im Repository `Settings` öffnen.
4. Links `Pages` auswählen.
5. Unter `Build and deployment` als Quelle `Deploy from a branch` auswählen.
6. Branch `main` und Ordner `/(root)` auswählen.
7. Auf `Save` klicken.

Nach einigen Minuten erscheint die Website ungefähr unter:

`https://DEIN-GITHUB-NAME.github.io/mien-plaut/`

Wichtig:
- `index.html` muss im Hauptverzeichnis liegen.
- Den Ordner `images` vollständig mit hochladen.
- Die Datei `.nojekyll` nicht löschen.
